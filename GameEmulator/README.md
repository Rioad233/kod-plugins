# kod-plugins

#### 介绍
kod的全能游戏插件

#### 软件架构
软件架构说明  
1. 基于js模拟器实现  
2. 支持全部的nes游戏  
3. 支持历年来任天堂,世嘉的大部分游戏  
4. 支持街机  
5. 总游戏类型达24种  

#### 安装教程

1.  将目录拷贝到kod的plugins  
2.  去插件中心找到该插件启用即可
